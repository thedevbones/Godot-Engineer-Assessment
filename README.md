# Project Setup Instructions

## How to Run the Project

1. **Clone the repository**
	- Use `git clone` to download the project files

2. **Import in Godot 4.6**
	- Open Godot Engine version 4.6
	- Select "Import" and navigate to the cloned repository folder
	- Import the project

3. **Play**
	- Once imported, press the Play button or use `F5` to run the project
